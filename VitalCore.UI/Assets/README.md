# VitalCore Design Assets

Placeholder structure for premium hardware monitor visuals. Generate PNGs with these specs and replace.

## Logo Principal
- **Dimensiune**: 1024x1024
- **Prompt/Stil**: Scut minimalist, linie puls neon cyan, fundal dark slate, stil sticlă (glassmorphism).

## Splash Screen
- **Dimensiune**: 620x300
- **Prompt/Stil**: Versiunea orizontală a logo-ului pe fundal transparent, cu textul &quot;VitalCore&quot; în fontul Segoe UI Variable.

## Store Icon
- **Dimensiune**: 50x50
- **Prompt/Stil**: Doar simbolul scutului, fără text, pentru vizibilitate maximă.

Salvează ca PNG transparente în acest folder. Folosește VS Package.appxmanifest &gt; Generate pentru scalări.
